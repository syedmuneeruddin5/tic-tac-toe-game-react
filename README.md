# Tic Tac Toe Game

A responsive, interactive Tic Tac Toe game built with React and Vite. This two-player game allows friends to play against each other in the classic strategy game.

## Features

- Two-player gameplay (Player 1: X, Player 2: O)
- Score tracking for both players
- Turn indicators
- Win/draw detection
- Responsive design
- Game reset functionality

## Getting Started

### Prerequisites

- Node.js (v14 or higher recommended)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/syedmuneeruddin5/Tic-Tac-Toe-Game-in-React
   cd Tic-Tac-Toe-Game-in-React
   ```

2. Install dependencies:
   ```bash
   npm install
   # or
   yarn
   ```

### Running the Application

1. Start the development server:
   ```bash
   npm run dev
   # or
   yarn dev
   ```

2. Open the server and play the game.

### Building for Production

To create a production build:

```bash
npm run build
# or
yarn build
```
